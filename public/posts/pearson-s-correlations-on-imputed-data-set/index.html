# required packages
require(corrplot)

library(Hmisc)

Introduction

Prior to performing principal component analyses, bivariate correlations need to be performed between variables to determine any underlying relationships. This is because PCA indentifies components that may better explain relationships or variability among variables that are related. If none of the variables are correlated, it would be futile to perform a PCA.

Discussion

The correlation plots show very strong relationships between the left and right measurements for the same test, as expected. It is also evident that negative correlations exist between axial length and corneal curvature (keratometry), while there are positive correlations between axial length and anterior chamber depth, and corneal thickness and intraocular pressure. These relationships are expected as they overlap in function. Interestingly, there are very weak negative correlations between corneal curvature (keratometry / K-values) and cup-to-disc ratio. This has been reported in other studies (https://www.nature.com/articles/eye201098.pdf?origin=ppub), although the biological reason for this relationship is not well-established. There is also a very weak positive correlaton between the intraocular pressure for left eye, but not right, measurements and cup-to-disc ratio measurements for both eyes. And weak negative correlations between the anterior chamber depth for the right eye, but not left, and cup-to-disc ratio for both eyes.
